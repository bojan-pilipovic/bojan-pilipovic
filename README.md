## Live Projects

### WalletRep — Onchain Wallet Reputation Scoring
[wlltrep.xyz](https://www.wlltrep.xyz/) · [Repo](https://github.com/KaizenBanjaLuka/walletrep) · [Kanban Board](https://github.com/orgs/KaizenBanjaLuka/projects/1)

A multichain dApp that scores and grades EVM wallet behavior across 5 chains. Wallets are scored across 6 dimensions and assigned a reputation tier from 1–100.

**What I built:**
- Defined the scoring model and grading logic (6 weighted dimensions, 5 tiers)
- Directed full-stack build: Node.js/Express backend, Vanilla JS frontend
- Designed the API architecture — rate limiting, in-memory caching, input validation
- Shipped shareable score pages with OG/Twitter meta tags
- Shipped embeddable SVG badge endpoint (`/badge/:wallet`)
- Deployed to production on custom domain via Vercel + Namecheap
- Maintained clean security practices — no wallet data stored, API key in environment

**Stack:** Node.js · Express · Alchemy API · Vanilla JS · Vercel  
**Roadmap:** Polygon · BNB Chain · Solana · Public API · Embed widget · Monetization

### wlltresume — Onchain Resume Builder
[wlltresume.xyz](https://www.wlltresume.xyz/) · [Repo](https://github.com/KaizenBanjaLuka/walletresume) · [Kanban Board](https://github.com/orgs/KaizenBanjaLuka/projects/2)

A Web3 identity dApp that generates a shareable onchain resume from any EVM wallet address or ENS name. No signup. Nothing stored. Everything computed live from public onchain data.

**What I built:**
- Defined the resume data model and reputation scoring algorithm (5 dimensions, log-scale scoring)
- Directed full-stack build: Next.js 14 App Router, custom CSS, no UI libraries
- Designed the Alchemy data layer — multichain queries, protocol detection via contract address mapping, NFT portfolio aggregation
- Shipped dynamic OG image generation for rich social previews on Twitter/Discord
- Deployed to production on custom domain (wlltresume.xyz) via Vercel + Namecheap
- Cross-links with WalletRep — complementary products in a growing Web3 identity suite

**Stack:** Next.js 14 · Alchemy API · ethers.js · Vercel  
**Roadmap:** Connect Wallet flow · PDF export · DAO detection · Wallet Passport integration
